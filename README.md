# AstrBot 更好用的帮助菜单

纯文字自定义帮助菜单插件 —— 告别图片式 `/help`，在 WebUI 里填菜单，聊天里发 `/h` 原样展示。

## 仓库

https://github.com/balareshe/AstrBot-help

## 安装

1. 下载插件 zip
2. 解压到 AstrBot 的 `data/plugins/` 目录
3. WebUI 中启用插件即可，无需额外依赖

## 使用

### 命令

| 命令 | 说明 |
|------|------|
| `/h` | 显示自定义菜单 |
| `/帮助` | 同 `/h` |

### 配置菜单

在 WebUI 插件面板中找到 **better_help**，在「菜单内容」文本框中填写你的菜单，每行一条，保存后发送命令即可看到效果。

示例配置：

```
🤖 /h - 查看本菜单
🤖 /帮助 - 同 /h
🎮 /roll - 掷骰子
😆 /joke - 讲个笑话
📊 /status - 机器人状态
```

填什么就展示什么，格式完全由你决定。

## 特点

- 纯文字输出，不依赖图片渲染
- 零依赖，即装即用
- WebUI 直接编辑菜单，无需改代码或手动编辑文件
- `/h` 短命令，不与内置 `/help` 冲突
