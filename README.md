# AstrBot 更好用的帮助菜单

> 📢 **官方 QQ 交流群：1107042979**，欢迎大家一起交流！

纯文字自定义帮助菜单插件 —— 告别图片式帮助，**支持自定义命令名**，在 WebUI 里填什么就发什么。

## 仓库

https://github.com/balareshe/AstrBot-help

## 安装

1. 下载插件 zip
2. 解压到 AstrBot 的 `data/plugins/` 目录
3. WebUI 中启用插件即可，无需额外依赖

## 使用

### 默认命令

`/h` 和 `/帮助` 始终可用，展示默认菜单。

### 自定义命令（核心功能）

在 WebUI 插件面板中找到 **better_help**，点击 `+` 添加自己的命令：

| 配置项 | 说明 |
|--------|------|
| 命令名 | 自定义触发命令，如填 `fun` → 用 `/fun` 触发 |
| 菜单内容 | 触发时原样展示的文字 |

示例：命令名填 `功能`，菜单内容写 "🎮 /roll - 掷骰子"，用户发 `/功能` 就展示这段内容。

你可以添加无数个自定义命令，每个命令对应独立菜单，互不影响。

## 特点

- 纯文字输出，不依赖图片渲染
- **支持自定义命令名**，想叫什么名字就叫什么
- 零依赖，即装即用
- `/h` 短命令，不与内置冲突
